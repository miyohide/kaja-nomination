# Kaja Nomination

WEB applications for voting RubyKaja.

## Usage
```
# You should install Bundler version 1.2.0.pre.1
$ gem install bundler --pre
$ git clone git@github.com:tyabe/kaja-nomination.git
$ cd kaja-nomination
$ bundle install --without production
$ padrino rake seed
$ padrino s
```
